# MD5 Collision Analyzer

A Python application that demonstrates MD5 hash generation and comparison using sample files. This project was developed to understand the behavior of MD5 hashing and illustrate the concept of hash collisions in an educational setting.

## Features

- Generate MD5 hashes
- Compare file hashes
- Analyze collision examples
- Simple Python implementation
- Educational demonstration

## Technologies Used

- Python
- hashlib
- Tkinter (if your project uses it)

## Project Structure

```
collision_files/
md5.py
md5-1.gif
md5-2.gif
report.txt
```

## How to Run

```bash
python md5.py
```
